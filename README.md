OS dependencies:
* Node > 16.x
* FFMPEG

Package dependencies:
* discord.js@14
* discord-player

Need to create an Discord Application for the BotToken 

To install use:
```shell
git clone https://github.com/iFraan/fraans-discord-music.git
npm install
npm run start
```
